# Proyecto MongoDB - Sistema de Tienda Online

## Descripción
Este proyecto implementa una base de datos MongoDB para un catálogo de productos de tienda online.

## Estructura
- `database/`: Datos iniciales y configuración
- `queries/`: Consultas MongoDB organizadas  
- `docs/`: Documentación del proyecto

## Consultas Implementadas
- Operaciones CRUD básicas
- Filtros avanzados
- Consultas de agregación
